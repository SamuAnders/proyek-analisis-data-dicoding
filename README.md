# proyek-analisis-data-dicoding
## Set Up Environment Anaconda
- conda create --name main-ds python=3.12
- conda activate main-ds
- pip install -r requirements.txt
## Set Up Environment CMD
- cd YOURPATH
- mkdir venv
- cd venv
- pip install -r requirements.txt
- ./Scripts/activate
## Run Streamlit
streamlit run dashboard.py
